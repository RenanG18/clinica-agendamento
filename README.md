# Sistema de Agendamento de Clínica

## Tecnologias
- Vue.js
- Node.js
- MongoDB

## Funcionalidades
- Cadastro e login
- Agendamento de consultas
- Consulta de CEP
- Previsão do clima
- Painel admin

## Como rodar o projeto

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
