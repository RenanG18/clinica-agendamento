# Sistema de Agendamento de Clínica

## Tecnologias
- Vue.js
- Node.js
- MongoDB

## Funcionalidades
- Cadastro e login
- Agendamento de consultas
- Consulta de CEP
- Previsão do clima
- Painel admin

### Banco de Dados:
- MongoDB Atlas

### Outros:
- JWT (autenticação)
- bcrypt (criptografia de senha)
- ViaCEP (consulta de endereço)
- OpenWeatherMap (previsão do tempo)
  
## Como rodar o projeto

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
